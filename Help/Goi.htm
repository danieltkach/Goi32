Goi

 

I created this program because I wanted to learn kanji and all their readings. But then I started using it to study vocabulary lists. I think it serves both purposes well. I just hope you find it useful.

This is what you will see when Goi starts:

 

 

 

To start using the program you will need four text files in shift-jis format.

  1. The words written in real Japanese, I mean, using kanji.
  2. The meaning of the words in the first file.
  3. The readings of the kanji or the word written just in hiragana (or romaji).
  4. The readings in romaji (this file is essential).

 

In order to load those lists into the program you must use the second button on the toolbar. When you click on it a menu appears and you can load the files one by one by choosing the right option. After loading all the lists, you are asked to save a .goi file. You will load this .goi file the next time instead of loading the lists one by one. The four files you load will be moved into a new folder with the same name as the .goi file; the program must do this for everything to work properly.

 

 

You will also need to enter a description for the file. You will see it on the main window’s title bar when you load the .goi file. If you made a mistake you can edit the description in the .goi file. Anyway, it is not recommended to create a .goi file manually (you may forget to add the extra line that corresponds to the file where the information about the test will be logged…). Just so you know, the first line is the description of the file (what will appear on the title bar), the second, third, fourth and fifth, are the lists for words in kanji, meaning, kana reading, and romaji reading respectively; and the last line is what I said before (log file).

 

 

 

Then you click on “Mix lists” and you can start testing the words or kanji in different ways. Choose one word from each list so that they match and then right click (you don’t need to check the boxes, use the boxes if you find any use for them). If you are right the three lines selected will disappear, and the window will turn green; if you are wrong the window will turn red.

 

 

 

Basically there are at least 6 different ways of testing the lists. You just need to click on any of these buttons:

 

 

If you press K-M (kanji to meaning) this window will appear:

 

 

On the black panel you will see the words that are loaded in the first list, and you are asked to write the meaning, that is, the words in the second list. The K-R button works the same, but you have to write the readings for the kanji or the word that appears on the black panel.

 

The M-K and R-K buttons offer two new options: a multiple choice exercise, and a draw the kanji exercise.

 

 

 

If you click on “Multiple choice” you will see this window:

 

 

Just click on the kanji corresponding to the reading or the meaning that is shown on the black panel. You also have these three buttons:

 

                                                                                                                                                    

If you want the kanji to disappear when you are right just click on the first one. If the second one is activated, the right answer will be highlighted if you are wrong. If the third button is down, Goi will wait till you are right.

 

Then you have the Draw the kanji option. If you choose this option from the pop up menu this window will appear:

 

 

Just draw the kanji before clicking on “Show”, then you must be honest and say whether you were right or wrong by clicking on “Correct!” or “Wrong!”. If you check the “Help” box you will see the reading if you are being shown the meaning, or you will see the meaning if you are being shown the reading. Try to use this option if it is really necessary, for example when there are two or more kanji with the same reading or with the same meaning.

 

After finishing any of the tests described above you will see a window that will ask you to “create a new item” or to “update results”. Goi keeps track of the word lists you are studying and their current average. By choosing “create new item” a new entry with its corresponding average will be stored; you can also update the average for the entry if the word list was tested before.

 

 

 

You will also see which ones you did well and in which ones you were wrong.

If you click on the Wrong Answers button on the window shown below you can save a new .goi file which will store the wrong answers to be practiced again. I recommend to save this file in the wrong answers folder. An entry is also added to a list has all the wrong answers files that were saved, and you can see these entries by clicking on this button   on the tool bar.

 

 

The entries are shown on this window:

 

Each file may have very few items to study so what you can do is join several of them and save them with a new name. You do this by selecting the entries (checking the boxes) and then clicking on the “Join” button.

 

Another option you might find interesting (or annoying!) is the possibility of being asked all the words you didn’t do well after a certain number of right guesses. You just need to check “Restest mistakes” from the “Options” menu:

 

 

You have a couple of more options, like the possibility of studying hiragana and katakana; they’re very simple to use and don’t need explanation.
